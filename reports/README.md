# Reports PDF Storage

This folder stores PDF copies of crypto research reports.

## How to Add PDFs

1. Download the PDF from the source
2. Name it descriptively: `source-report-name-year.pdf`
3. Add entry to `js/reports.js` with `pdfPath: "reports/your-file.pdf"`

## Example
```
messari-crypto-theses-2025.pdf
a16z-state-of-crypto-2024.pdf
chainalysis-crypto-crime-2024.pdf
```
